# Frontend Service

**Language:** Node.js (Express)
**Port:** 8080

## Description
A modern, responsive dashboard serving as the entry point for the polyglot e-commerce system.

## Features
- Glassmorphism Design
- Responsive Grid
- Visualization of backend microservices
- **User Authentication**: Login/Signup (Simulated)
- **Shopping Cart**: Fully functional cart flow
- **Admin Portal**: Add products dynamically

## How to Run
```bash
npm install
npm start
```

Visit `http://localhost:8080` in your browser.
